# 摩天轮票务 · 小红书舆情管理工具

针对小红书平台的品牌舆情监控系统，支持批量分析笔记风险、自动生成举报文案、一键跳转投诉。

---

## 工作流程

```
插件采集笔记 → 导出 Excel 到监听文件夹 → 自动上传分析 → 查看结果 → 一键投诉
```

1. 用「社媒助手」插件采集关键词笔记，导出 Excel 到桌面 `摩天轮舆情管理/` 文件夹
2. `folder_watcher.py` 自动检测新文件并上传到服务
3. AI 分析每条笔记的风险等级、举报类型、举报文案
4. 在网页端查看排序结果，点"去投诉 ↗"直接跳转小红书，复制文案提交

---

## 快速启动

**1. 安装依赖**
```bash
pip3 install flask openai openpyxl xlrd watchdog requests python-dotenv
```

**2. 配置环境变量**

新建 `.env` 文件：
```
API_KEY=你的API密钥
API_BASE_URL=https://你的API地址/v1
```

**3. 启动服务**（终端一）
```bash
python3 app.py
```

**4. 启动文件夹监听**（终端二）
```bash
python3 folder_watcher.py
```

**5. 打开浏览器**

访问 http://localhost:5001

## 风险分级

| 等级 | 说明 |
|------|------|
| 高风险 | 假冒官方、盗用品牌诈骗、严重不实指控 |
| 中风险 | 明确负面评价、疑似冒用品牌、引导私下交易 |
| 低风险 | 轻微吐槽，情绪温和，影响力有限 |
| 无风险 | 客观中性讨论或正面评价 |

结果按「风险等级 → 影响力分数」双维度排序，高风险高影响力的笔记优先展示。

---

## 举报类型

AI 会从以下类型中自动选择最合适的一个：

谩骂嘲讽/对立、虚假不实、标题党、引人不适、诱导关注点赞、搬运抄袭他人作品、侵犯权益、色情低俗、政治敏感、涉嫌诈骗、种族歧视、违法违规、违反公德秩序、危害人身安全、涉未成年不当内容、违规营销/假货、导流到站外

---

## 影响力计算公式

```
影响力分数 = 点赞量 + 收藏量×2 + 评论量×3 + 分享量×4
高影响力 > 1000 | 中影响力 ≥ 300 | 低影响力 < 300
```

---

## Excel 格式要求

上传的 Excel（`.xls` 或 `.xlsx`）必须包含以下列：

`笔记标题`、`笔记内容`、`笔记话题`、`点赞量`、`收藏量`、`评论量`、`分享量`

有 `笔记链接` 列时，结果页会显示"去投诉 ↗"按钮。
